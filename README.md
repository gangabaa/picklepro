# PicklePro App - Revenue Plan

## How It Makes Money

### 1. Affiliate Marketing (IMMEDIATE)
- Amazon affiliate links for paddles/gear
- Commission: 3-10% per sale
- Top paddles: $80-170 → $2.40-$17 per sale

### 2. Premium Subscription ($9.99/month)
- AI Coach analysis
- Personalized training plans
- Video form analysis
- **Projected: 500 free users → 5% convert = 25 subs × $9.99 = $250/month**

### 3. In-App Ads
- Banner ads (Google AdMob)
- Interstitial ads
- **Projected: $50-100/month with 50K downloads**

### 4. Court Booking Fees (FUTURE)
- Partner with courts
- Take 5-10% booking fee
